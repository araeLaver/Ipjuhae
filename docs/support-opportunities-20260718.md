# Rentme 7월 18일 사업기회/지원사업 발굴 메모

용도: [DOW-473](/DOW/issues/DOW-473) 일일 실행 산출물

## 우선순위 요약

1. K-DATA AI·데이터 문제해결은행 맞춤 지원: 계속 1순위. K-DATA 공식 공지 기준 2026-07-03부터 2026-07-21까지 참여기업을 모집하며, 중소·소상공인 및 예비창업자의 데이터·AI 문제정의와 전문가 지원을 다룬다. Rentme의 "임대 신뢰 리포트 설명 가능성"과 개인정보 없는 synthetic event 설계에 가장 직접적으로 맞다.
2. 공공데이터 활용기업 AI연계 맞춤형 성장 지원사업: 2순위 신규 제출 후보. 공공데이터포털 공지 기준 모집기간은 2026-07-15 10:00부터 2026-08-14 18:00까지이며, 공공데이터·AI 활용 예정 기업 또는 예비창업자를 대상으로 한다. Rentme는 전월세 실거래가, 행정동 생활권, 교통·편의시설, 주거복지 데이터와 결합한 주거 신뢰 리포트로 포지셔닝할 수 있다.
3. KISA ICT 중소기업 정보보호 지원사업 수요기업: 조건부 유지. KISA 공지 기준 2026-06-26에 수요기업 모집이 올라왔고, 정보보호 컨설팅, IT 보안 패키지, SECaaS 이용 지원 성격이다. Rentme의 개인정보·증빙·접근로그 리스크와 맞지만, 중소기업 요건·사업장 소재지·자부담 승인 확인 전에는 신청 작업으로 넘기지 않는다.
4. 정부 첫 실증·구매 프로젝트(스마트도시): 빠른 No-Go/조건부 판정 필요. 창업진흥원 목록 기준 마감은 2026-07-22 15:00이다. 주거 신뢰·임대 관리·입주자 검증 수요과제가 있으면 실증 레퍼런스 후보지만, 수요과제 불일치 시 K-DATA 준비를 방해하므로 신청서 작성 전 적합성만 확인한다.
5. 벤처확인 인증준비기업 맞춤형 무료 진단 지원사업: 낮은 우선순위 검토. K-Startup 목록 기준 마감 여유가 2026-07-31 전후로 보이며, 법인/사업자 상태와 기술성 자료가 준비되어 있을 때만 후속 검토 가치가 있다. 현재는 예비창업자/사업자 상태 확인 전까지 대기 후보로 둔다.

## 후보별 실행 정보

### 1. K-DATA AI·데이터 문제해결은행 맞춤 지원

- 마감: 2026-07-21 화요일.
- 기대효과: 데이터·AI 관점의 문제정의, 데이터 목표 설정, 분석·활용 컨설팅을 통해 Rentme의 리포트 설명 가능성과 고객개발 funnel을 정리한다.
- 필요한 자료: 신청 주체, K-DATA 계정, 팀/기업 표기명, 공개 연락처, 사업자 또는 예비창업자 정보, 소재지, 첨부 허용 범위, 최종 제출자.
- 다음 행동: 보드 확인값이 들어오면 [DOW-424](/DOW/issues/DOW-424)에서 최종 신청서 입력 문안으로 전환한다.

### 2. 공공데이터 활용기업 AI연계 맞춤형 성장 지원사업

- 마감: 2026-08-14 금요일 18:00.
- 기대효과: 공공데이터 기반 PropTech/주거 신뢰 서비스 포지션을 만들고, K-DATA 이후 제출 가능한 2순위 후보를 확보한다.
- 필요한 자료: 활용 공공데이터 목록, AI 적용 범위, 예비창업자 또는 기업 신청 자격, 이메일 제출 양식, 개인정보 없는 mock 리포트 예시.
- 다음 행동: Rentme 적합성 메모와 신청 초안 목차를 만든다.

### 3. KISA ICT 중소기업 정보보호 지원사업 수요기업

- 마감: 예산 소진 또는 공고상 기간 종료 전까지로 관리. 오늘 기준 KISA 수요기업 모집 공고는 진행 중으로 확인했다.
- 기대효과: 개인정보 처리, 증빙 업로드, 동의 기반 공유 링크, 관리자 접근로그를 외부 보안 기준으로 점검한다.
- 필요한 자료: 중소기업 해당 여부, 사업장 소재지, 지역정보보호지원센터 권역, 자부담 20% 및 VAT 승인 여부, 적용하려는 패키지 유형.
- 다음 행동: 보드 확인값 4개가 들어오기 전까지 blocked 후보로 유지한다.

### 4. 정부 첫 실증·구매 프로젝트(스마트도시)

- 마감: 2026-07-22 수요일 15:00.
- 기대효과: 수요과제가 맞으면 지자체/공공기관 실증 레퍼런스와 초기 구매 가능성을 검토할 수 있다.
- 필요한 자료: 수요기관별 스마트도시 과제 목록, 업력 요건, 사업자 상태, Rentme를 공공 스마트도시 문제로 설명할 수 있는 범위.
- 다음 행동: 수요과제 목록만 확인해 2026-07-19까지 Go/No-Go를 확정한다. 신청서 작성은 Go 판정 후에만 시작한다.

### 5. 벤처확인 인증준비기업 맞춤형 무료 진단 지원사업

- 마감: K-Startup 목록상 2026-07 말로 추정되며 세부 공고 확인 필요.
- 기대효과: Rentme가 기술기반 서비스로 설명 가능한지 외부 진단을 받을 수 있고, 향후 지원사업 가점/신뢰자료를 준비할 수 있다.
- 필요한 자료: 법인 또는 사업자 상태, 기술성 설명, 서비스 구현 화면, 지식재산/알고리즘 설명 범위.
- 다음 행동: K-DATA와 공공데이터 후보보다 뒤로 두고, 법인/사업자 상태가 확인되면 세부 공고를 재확인한다.
