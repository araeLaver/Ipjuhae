## 0. Go/No-Go 권고

### 결론

Rentme는 제32기 POSCO IMP 투자트랙에 **조건부 Go**로 판단합니다. AI/데이터 기반 PropTech와 주거 신뢰 데이터 포지션은 모집분야의 `AI/로봇`, `에너지/인프라`, `미래 유망분야`에 제한적으로 맞출 수 있고, Seed 단계라면 Part A 후보로 제출 가능합니다.

다만 POSCO그룹 직접 협업성은 국토교통/NEXT 챌린지보다 약합니다. 제출 문안은 "포스코그룹 주거·도시·인프라 접점에서 검증 가능한 임대 신뢰 데이터 파일럿"으로 좁히고, 제철/소재/제조 직접 협업처럼 과장될 수 있는 표현은 제외해야 합니다.

## 1장 티저

### 회사/서비스

**Rentme(임주해)** 는 세입자가 먼저 동의 기반 신뢰 프로필을 만들고, 집주인·임대관리자가 조건에 맞는 세입자와 대화를 시작할 수 있게 돕는 주거 임대 매칭 플랫폼입니다.

기존 부동산 서비스가 매물 검색과 광고 중심이라면 Rentme는 세입자 프로필, 동의 기반 인증, 레퍼런스, 매칭 참고 지표, 집주인 검색 UX를 결합해 임대차 시장의 정보 비대칭을 줄입니다.

### 문제

- 집주인은 계약 전 세입자의 납부 안정성, 생활 패턴, 이전 임대인 레퍼런스를 구조적으로 확인하기 어렵습니다.
- 세입자는 원하는 조건을 반복 입력하며 여러 플랫폼을 순회하지만, 본인의 신뢰도와 선호 조건을 협상 자산으로 쓰기 어렵습니다.
- 허위매물, 불투명한 중개 과정, 전세사기 우려가 커지면서 양측 모두 검증 가능한 신뢰 데이터가 필요합니다.

### 솔루션

- 세입자: 직업/소득/거주 선호/생활 패턴/본인 확인/이전 임대인 레퍼런스를 기반으로 신뢰 프로필을 생성합니다.
- Rentme: 동의 기반 데이터와 레퍼런스를 매칭 참고 지표와 리포트로 정리하고, 세입자-매물 적합도를 계산합니다.
- 집주인: 조건에 맞는 세입자를 검색하고, 관심 세입자에게 먼저 제안하거나 대화를 시작합니다.

### 시장

- 한국 전월세 임대 거래는 연간 약 200만건 규모로 추정됩니다.
- 수도권 및 광역시 중심의 서비스 가능 시장은 연간 약 120만건으로 보고, 초기에는 서울 주요 생활권의 임차인/임대인 매칭으로 진입합니다.
- 글로벌 PropTech 및 AI 기반 주거 서비스 투자가 확대되는 흐름 속에서, Rentme는 "신뢰 기반 임대 매칭"이라는 좁은 문제에 집중합니다.

### 비즈니스 모델

- 세입자 Plus 구독: 월 9,900원, 무제한 매칭 요청과 프로필 배지/우선 노출
- 집주인 Pro 구독: 월 29,900원, 세입자 풀 열람과 고급 필터
- 임대관리회사 Business: 월 99,000원, 다주택 관리 및 API 연동
- 거래/부가 수익: 계약 성사 수수료, 프로필 인증 리포트, 전입신고/확정일자 지원, 분쟁 예방 컨설팅

### 현재 상태

- MVP 핵심 범위 구현 진행: 세입자 프로필, 집주인 세입자 검색, 매물 등록, 매칭, 메시지, 레퍼런스 요청, 매칭 참고 지표 재계산
- 운영 준비 문서화: `docs/LAUNCH_CHECKLIST.md`, `docs/MVP_SMOKE_QA.md`
- 기술 스택: Next.js, TypeScript, PostgreSQL, Docker, Stripe, OpenAI 연동 옵션
- 예정: 서울 강남/마포 중심 베타 파일럿, 초기 세입자 모집 후 집주인/임대관리자 세일즈
- 제출 전 미확정: 법인명, 대표자, 설립일, 기존 투자 여부, 실제 사용자/대기자/파일럿/매출 지표, 공개 데모 URL

### POSCO IMP 협업 가설

- 포스코그룹의 주거·도시·인프라 접점에서 임대 신뢰 데이터 기반 파일럿을 검토할 수 있습니다.
- Ground 서울/포항/광양 입주 또는 보육 인프라와 연계해 지역 기반 주거 매칭 실험을 진행할 수 있습니다.
- AI/데이터 기반 임대 매칭 리포트와 도시 주거 데이터 분석을 확장하면 스마트시티·인프라 운영 의제와 제한적으로 연결할 수 있습니다.
- 협업 표현은 "검토 가능성"과 "파일럿 가설" 수준으로 한정하고, 포스코그룹의 현업 수요가 확인된 것처럼 쓰지 않습니다.

### 지원서 요약 문안

Rentme는 세입자가 먼저 동의 기반 신뢰 프로필을 만들고, 집주인·임대관리자가 조건에 맞는 세입자와 대화를 시작할 수 있게 돕는 주거 임대 매칭 플랫폼입니다. 매물 광고 중심의 기존 부동산 서비스와 달리 세입자의 입주 가능일, 예산, 거주 선호, 직업/소득 관련 자료, 이전 임대인 레퍼런스를 구조화해 임대차 거래 전 단계의 탐색·검증·대화 비용을 줄입니다.

Rentme는 AI/데이터 기반 프로필 요약과 매칭 참고 지표를 활용해 임대차 시장의 정보 비대칭을 줄이고자 합니다. 해당 지표는 신용평가나 법적 자격 판단이 아니라, 사용자가 동의한 범위의 자료를 바탕으로 대화를 돕는 참고 정보로 운영합니다. 초기에는 서울 주요 생활권에서 세입자 모집과 집주인/임대관리자 파일럿을 진행하고, 이후 주거 데이터 기반 리포트와 임대관리회사 Business 요금제로 확장합니다.

POSCO IMP에서는 포스코그룹의 주거·도시·건설·인프라 접점과 연계해 임대 신뢰 데이터 기반 파일럿 가능성을 검토하고자 합니다. Ground 서울/포항/광양 보육 인프라와 투자·멘토링을 활용해 개인정보·임대차·차별 리스크를 낮춘 설명 가능한 PropTech 모델로 고도화하겠습니다.

- 실제 사용자, 대기자, 파일럿, 매출, 제휴, 투자, 특허, 상표, 수상 지표 중 공개 가능한 항목
- MVP 구현 범위 중 외부 심사자가 직접 확인 가능한 기능
- 개인정보, 레퍼런스, 매칭 참고 지표 설명의 공개 가능 범위

### 법률/표현 확인

- 매칭 참고 지표가 신용평가, 자동 의사결정, 임차인 차별로 오인되지 않도록 문구 검토
- 소득/직업/레퍼런스 데이터 수집은 사용자 동의 기반이라는 점 명시
- 집주인 검색/제안 기능은 차별적 선별이 아니라 사용자가 공개한 선호와 조건 기반 대화 시작 도구로 설명

### 제출 운영 확인

- POSCO IMP 홈페이지 계정 접근 권한
- 제32기 POSCO IMP 신청 화면 노출 여부 확인
- 제32기 POSCO IMP 사업계획서 양식 다운로드 및 목차 확인
- 온라인 입력 필드 글자 수 제한, 첨부 파일 형식/PDF 용량 제한
- 사업계획서, 사업자등록증, 법인등기부등본, 투자확인서 또는 투자 내역 증빙 파일 준비
