## 3. 문제 정의 (Problem)

### 현재 임대 시장의 3대 페인포인트

1. **정보 비대칭**: 집주인은 세입자 신뢰도를 사전에 알 수 없음 → 깡통전세·보증금 사기 피해 급증
2. **공급자 중심 플랫폼**: 기존 플랫폼은 매물(집)이 중심 → 세입자는 수동적 검색자로 전락
3. **중개사 의존**: 불투명한 중개 과정, 과도한 수수료, 허위매물 문제

### 세입자 입장 고통
- 원하는 조건의 매물을 찾기 위해 수십개 플랫폼 순회
- 집주인/중개사에 대한 신뢰 부족
- 계약 후 분쟁(임차인 보호 제도 미흡)

### 집주인 입장 고통
- 신뢰할 수 있는 세입자 선별 어려움
- 허위 이력·연체 이력 사전 파악 불가
- 좋은 세입자를 먼저 확보하는 채널 부재

## 4. 솔루션 — Rentme의 역방향 매칭

### 핵심 메커니즘

```text
[세입자] → 프로필 생성 (직업, 소득, 생활패턴, 신용점수 동의)
          ↓
[Rentme AI] → 신뢰 스코어 산출 + 매칭 알고리즘
          ↓
[집주인] → 검증된 세입자 후보 수신 → 역제안(Reverse Offer)
          ↓
[매칭] → 계약까지 플랫폼 내 원스탑 처리
```

### 기존 플랫폼 대비 차별점
| 구분 | 직방/다방 | **Rentme** |
|------|---------|-----------|
| 매칭 방향 | 집(공급) → 세입자 | **세입자(수요) → 집** |
| 신뢰 검증 | 없음 | **AI 신뢰 스코어** |
| 집주인 역할 | 매물 등록 | **세입자 스카우트** |
| 허위매물 | 만성적 문제 | **구조적 제거** |
