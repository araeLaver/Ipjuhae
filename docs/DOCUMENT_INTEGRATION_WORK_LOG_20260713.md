- 근거·사실·파생결과 의존관계

- 거래단계별 최소공개
- 거래증명 기반 양방향 레퍼런스와 이상관계 격리
- 목적별 동의, 열람 감사, 이의제기와 보존기간 처리
- OCR 작업, 수동검수, 정책·모델 버전과 Outbox
- 중복 요청 방지, 요청 추적과 토큰 남용 차단

- 계약 전 확인 리포트 전용 흐름
- 임차인·임대인·주택 3축 확인상태표
- Trust Card 발급·회수·공개열람
- 문서 악성검사 완료 전 OCR 차단
- AI 처리·개인정보·사람검토 재현기록
- 고객검증 목표와 실제 실적 분리
- 법률·보안 승인 게이트
- 중개사·기관용 조직계정 기반
