# 7월 지원사업 제출 우선순위 결정표

작성일: 2026-07-16
대상: Rentme(입주해)

## 결론

이번 주 실제 제출 후보는 `K-DATA AI·데이터 문제해결은행 맞춤 지원` 1건으로 좁힌다.

K-DATA는 마감이 2026-07-21 18:00로 가장 임박했고, 예비창업자까지 신청 가능성이 열려 있으며, Rentme의 현재 산출물만으로도 개인정보 없는 문제정의, synthetic event, mock 화면 캡처 중심의 제출 패키지를 만들 수 있다.

KISA ICT 중소기업 정보보호 지원사업은 적합성은 높지만 신청 주체가 중소기업인지, 사업장 권역이 어디인지, 보안 패키지 또는 SECaaS 자부담 20%와 VAT를 승인할지 확인되기 전에는 제출 패키지로 넘기지 않는다.

중소기업 AI훈련확산센터는 현 시점 보류다. 예산 소진 전 상시성 성격이고, 신청 주체가 중소기업인지 확인되지 않았으며, 즉시 제출형 사업보다 컨설팅/훈련 성격이 강하다.

## 보드 확인값

K-DATA를 이번 주 제출 후보로 밀려면 아래 8개만 우선 답하면 된다.

1. 신청 주체: 법인, 개인사업자, 예비창업자 중 무엇으로 신청할지
2. K-DATA 계정: 기업회원 또는 예비창업자 계정 사용 가능 여부
3. 신청서 표기명: 대표자/담당자명과 공개 가능한 서비스명
4. 공개 연락처: 신청서에 넣을 이메일과 전화번호 범위
5. 사업자 또는 예비창업자 정보: 사업자등록번호 또는 예비창업자 정보 입력 가능 여부
6. 소재지: 신청서에 넣을 주소 또는 활동 지역
7. 첨부 범위: 개인정보 없는 mock 화면 캡처 3~5장과 내부 초안 문서 사용 가능 여부
8. 최종 제출자: 로그인, 신청서 작성, 최종 제출 버튼을 누를 사람

KISA는 K-DATA 제출 여부와 별도로 아래 4개가 확인될 때만 재개한다.

1. 신청 주체가 `중소기업기본법`상 중소기업에 해당하는지
2. 사업장 소재지와 해당 지역정보보호센터 권역
3. 보안제품 또는 SECaaS 도입 시 자부담 20% 및 VAT 승인 여부
4. 정보보호 컨설팅만 신청할지, 보안 패키지/SECaaS까지 염두에 둘지

AI훈련센터는 아래 3개가 확인될 때만 재검토한다.

1. 신청 주체가 중소기업인지, 예비창업 상태로도 문의할지
2. 소재지가 서울/경기/인천/강원 권역인지
3. 외부 훈련에 사용할 개인정보 없는 workflow 1개를 확정할 수 있는지
