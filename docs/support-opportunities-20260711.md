# Rentme 7월 소형 지원사업 검토 메모

작성일: 2026-07-11
용도: 스타트업 언론홍보 지원사업 및 특허출원·등록 비용 바우처 제출 여부 판단

## 결론

- 언론홍보 지원사업: 조건부 진행 권장. K-Startup 원문과 첨부 안내문 기준 이메일 제출형이며 별도 비용 청구 없음이 확인됐다. 다만 보도자료/이미지/대표 연락처 등 외부 제출자료는 보드 승인 전 발송하지 않는다.
- 특허출원·등록 비용 바우처: 조건부 보류 권장. 총사업비 500만 원 중 지원기관 부담금 250만 원과 관납료 별도 부담 구조라, 현 단계에서는 무료 바우처가 아니라 유료 IP 패키지에 가깝다.

## 1. 스타트업 언론홍보 지원사업

확인값:

- 공고명: `26년 7월 스타트업 언론 홍보 지원사업 참가사 모집 공고(1차)`
- 주관기관: 스타트업데일리
- 지역: 전국
- 접수기간: 2026-07-01 ~ 2026-07-15
- 원문 상세 URL: `https://www.k-startup.go.kr/web/contents/bizpbanc-ongoing.do?pbancSn=178326&schM=view`
- 신청 URL: 별도 신청 폼 없음. 이메일 접수 `contact@startupdaily.kr`
- 신청대상: 예비창업자 및 창업 7년 이내 초기 기업
- 제출서류: 첨부 양식의 보도자료 및 관련 이미지 파일
- 제출 방식: 이메일 접수. 이미지 파일은 HWP 내부 삽입이 아니라 별도 파일 첨부
- 비용/자부담: 스타트업데일리 홈페이지 보도자료 게시 지원, 별도 비용 청구 없음
- 문의: 전화 `02-564-0309`, 이메일 `contact@startupdaily.kr`, 홈페이지 `https://www.startupdaily.kr/`

리스크:

- 이미 배포된 보도자료 및 동일한 내용의 반복 배포는 불가하다.
- 단순 제품 홍보, 프로모션 홍보 등 과도한 광고성 문안은 게재 불가하다.
- 배포 이슈 예시는 투자 유치, 수상, 협약 체결, 공급계약 체결 등으로 제시되어 있다. Rentme는 베타 운영 준비/신규 서비스 런칭 관점으로 보수적 설명이 필요하다.
- 스타트업데일리는 스타트업 전문 매체 성격으로 보이며, 대형 종합지 배포가 아니라 매체 내 게시 지원으로 보는 것이 안전하다.

Rentme 제출 판단:

- 베타 운영 준비/신규 서비스 런칭 관점으로 제출 실익이 있다.
- 단, 보드 승인 전에는 외부 이메일 발송, 개인정보 입력, 이미지 파일 제출을 하지 않는다.
- 외부 검증, 금융 신용평가, 계약 보장처럼 보일 수 있는 표현은 제외해야 한다.
- `marketing/press-release-202607.md`의 보수적 제출 패키지를 기준으로 제출한다.

제출자료:

- 보도자료 1문단 요약 또는 본문
- 서비스 화면 이미지 3~5장
- 로고 이미지 1개
- 대표자명, 법인/사업자명, 공개 연락처, 베타 신청 URL
- 개인정보/테스트 데이터가 보이지 않는 이미지

보드 승인 요청안:

```md
스타트업데일리 7월 언론홍보 지원사업은 이메일 접수형이며, 스타트업데일리 홈페이지 보도자료 게시에 별도 비용 청구가 없다고 확인됐습니다. Rentme 베타 운영 준비 사실을 보수적 문구로 제출 준비해도 될지 승인 요청드립니다.

승인 필요값:
- 대표자명/기업명
- 공개 이메일 또는 연락처
- 베타 신청 URL
- 제출 가능한 로고 및 화면 캡처
- `신뢰점수`, `신용평가`, `검증 완료`, `계약 보장` 표현 제외 동의
- 이메일 접수처 `contact@startupdaily.kr` 발송 승인
```

## 2. 특허출원·등록 비용 바우처

확인값:

- 공고명: `2026년 특허출원·등록 비용 바우처 지원사업(8차) 시행계획 공고`
- 공고기관: (사)한국중소기업발전협회
- 접수기간: 2026-07-01 ~ 2026-07-17
- 지역: 전국
- 신청자격: 연구개발 활동을 수행하고 있는 스타트업 및 중소·중견기업
- 지원방식: 협회 지식재산위원회와 중소기업 전문변리사가 지원전담팀을 구성해 IP 진단, IP 포트폴리오 구상, 특허출원·등록 업무 지원
- 지원규모: 총사업비 500만 원, 협회 지원금 250만 원, 지원기관 부담금 250만 원
- 별도 부담: 지식재산처 관납료, 출원관납료, 우선심사신청 관납료, 등록관납료는 미포함이며 기업 부담
- 접수방법: 신청서 등 서류 일체 이메일 접수 `ip@ceo-korea.org`
- 제출서류: 사업자등록증, 회사소개서/사업계획서/IR자료 중 1개 이상
- 선정평가: 서면평가/전화인터뷰, 기술혁신성 50점과 사업성장성 50점

Rentme 제출 판단:

- IP 전략/특허 명세서 보강에는 맞지만, 최소 250만 원 이상의 자기부담이 있다.
- 관납료가 별도라 실제 현금 지출은 250만 원을 초과할 수 있다.
- 이미 법무/IP 제출용 공개/비공개 매트릭스가 있어 상담 준비는 가능하지만, 현재는 보드가 유료 진행 의사를 먼저 확인해야 한다.

조건부 진행 시 제출자료:

- 참가신청서
- 개인정보 동의서
- 사업자등록증
- 회사소개서, 사업계획서 또는 IR자료 1개 이상
- `docs/patent-public-private-matrix-202607.md` 기반 IP 설명 요약
- 외부 제출용 redacted 샘플만 포함하고 세부 산식, DB schema, OCR prompt, 실제 사용자 데이터는 제외

보드 승인 요청안:

```md
특허출원·등록 비용 바우처는 전국 대상이지만, 총사업비 500만 원 중 지원기관 부담금 250만 원과 관납료 별도 부담이 확인됐습니다. Rentme 특허/IP 패키지로 진행하려면 유료 진행 승인과 예산 상한을 먼저 정해야 합니다.

승인 필요값:
- 자기부담금 250만 원 및 관납료 별도 지출 가능 여부
- 출원 희망 범위: 임차인/임대인/주택 안전 리포트, 동의 기반 공개, 신뢰 참고 신호 중 우선순위
- 외부 변리사와 공유 가능한 자료 범위
```
